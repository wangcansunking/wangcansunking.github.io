第7课：多 Agent 协作 — AutoGen + CrewAI 范式

介绍

单个 Agent 有上下文窗口限制、单线程执行瓶颈，复杂任务需要多 Agent 协作。本课不依赖框架，手动实现多 Agent 协作系统：Agent 定义（角色、工具、约束）、消息传递协议、Orchestrator 分配任务、结果聚合、冲突解决，并对比 AutoGen 和 CrewAI 的设计思路。

遇到什么问题

单 Agent 处理"完整审查代码仓库并生成报告"时面临三大瓶颈：

flowchart LR
    P[单一 Agent] --> C1[上下文窗口限制\n代码总量超过 200K tokens]
    P --> C2[串行执行慢\nN 个模块顺序处理]
    P --> C3[单一视角偏差\n开发者 + 安全专家 角色冲突]

    C1 --> S[多 Agent 协作解决]
    C2 --> S
    C3 --> S

有哪些方案

方案 A：中央化 Orchestrator（主从模式）

// Orchestrator 分配任务给专业 Agent，收集结果
const orchestrator = new OrchestratorAgent();
const results = await orchestrator.dispatch(task, [analystAgent, securityAgent]);

✅ 控制流清晰，易于调试 ✅ Orchestrator 可做质量把关和冲突解决 ❌ Orchestrator 成为瓶颈，单点故障

方案 B：去中心化（Peer-to-Peer 消息传递）

// Agent 之间直接通信，无中央控制
agentA.send(agentB, { type: "REQUEST", payload: data });
agentB.on("REQUEST", async (msg) => { /* 处理并回复 */ });

✅ 无单点故障，可水平扩展 ❌ 消息路由复杂，调试困难

方案 C：Orchestrator + 并行执行（推荐）

// Orchestrator 分配，各 Agent 并行执行，结果聚合
const [r1, r2, r3] = await Promise.all([
  analystAgent.run(task),
  securityAgent.run(task),
  perfAgent.run(task),
]);
const report = await orchestrator.synthesize([r1, r2, r3]);

✅ 并行执行大幅缩短时间 ✅ 专业角色隔离，结果更准确 ❌ 需要设计好 Agent 接口和消息格式

怎么选择

选择方案 C：Orchestrator 负责任务分配和结果聚合，各专业 Agent 并行执行。这是 AutoGen GroupChat 和 CrewAI Process.parallel 的核心思路，不依赖框架也能实现。

怎么执行

Agent 定义（角色、工具、约束）

// src/multi_agent/agent.ts
import OpenAI from "openai";

const openai = new OpenAI({ apiKey: process.env.OPENAI_API_KEY });

export interface AgentMessage {
  from: string;
  to: string;
  type: "TASK" | "RESULT" | "REQUEST" | "RESPONSE" | "ERROR";
  payload: string;
  metadata?: Record<string, any>;
}

export interface AgentConfig {
  name: string;
  role: string;              // 角色描述
  goal: string;              // 目标
  constraints: string[];     // 约束条件
  tools?: string[];          // 可用工具列表
  model?: string;
}

export class Agent {
  public readonly name: string;
  private config: AgentConfig;
  private messageLog: AgentMessage[] = [];

  constructor(config: AgentConfig) {
    this.config = config;
    this.name = config.name;
  }

  async run(task: string, context?: string): Promise<string> {
    const systemPrompt = this.buildSystemPrompt();
    const userContent = context
      ? `背景信息：\n${context}\n\n任务：${task}`
      : `任务：${task}`;

    const response = await openai.chat.completions.create({
      model: this.config.model || "gpt-4o",
      messages: [
        { role: "system", content: systemPrompt },
        { role: "user", content: userContent },
      ],
      temperature: 0.3,
    });

    return response.choices[0].message.content || "";
  }

  private buildSystemPrompt(): string {
    const constraintsList = this.config.constraints
      .map((c) => `- ${c}`)
      .join("\n");
    const toolsList = this.config.tools?.join("、") || "无特定工具";

    return `你是 ${this.config.name}，角色：${this.config.role}

目标：${this.config.goal}

约束条件：
${constraintsList}

可用工具：${toolsList}

请严格按照你的角色和约束执行任务，不要越界处理其他角色负责的内容。`;
  }

  logMessage(message: AgentMessage): void {
    this.messageLog.push(message);
  }

  getMessageLog(): AgentMessage[] {
    return this.messageLog;
  }
}

消息传递协议

// src/multi_agent/message_bus.ts

export interface Message {
  id: string;
  timestamp: number;
  from: string;
  to: string | "broadcast";
  type: "TASK" | "RESULT" | "QUERY" | "RESPONSE" | "ERROR";
  payload: string;
  replyTo?: string;   // 引用的消息 ID
}

type MessageHandler = (message: Message) => Promise<void>;

export class MessageBus {
  private subscribers: Map<string, MessageHandler[]> = new Map();
  private messageLog: Message[] = [];

  subscribe(agentName: string, handler: MessageHandler): void {
    const handlers = this.subscribers.get(agentName) || [];
    handlers.push(handler);
    this.subscribers.set(agentName, handlers);
  }

  async publish(message: Omit<Message, "id" | "timestamp">): Promise<void> {
    const fullMessage: Message = {
      ...message,
      id: `msg_${Date.now()}_${Math.random().toString(36).slice(2, 8)}`,
      timestamp: Date.now(),
    };

    this.messageLog.push(fullMessage);

    if (message.to === "broadcast") {
      // 广播给所有订阅者
      const allHandlers = Array.from(this.subscribers.values()).flat();
      await Promise.all(allHandlers.map((h) => h(fullMessage)));
    } else {
      const handlers = this.subscribers.get(message.to) || [];
      await Promise.all(handlers.map((h) => h(fullMessage)));
    }
  }

  getLog(): Message[] {
    return this.messageLog;
  }
}

Orchestrator（任务分配 + 结果聚合）

// src/multi_agent/orchestrator.ts
import OpenAI from "openai";
import { Agent } from "./agent";
import { MessageBus } from "./message_bus";

const openai = new OpenAI({ apiKey: process.env.OPENAI_API_KEY });

export interface TaskAssignment {
  agentName: string;
  subtask: string;
  priority: "high" | "medium" | "low";
  dependsOn?: string[];  // 依赖的其他 Agent 名称
}

export class Orchestrator {
  private agents: Map<string, Agent> = new Map();
  private bus: MessageBus;

  constructor(bus: MessageBus) {
    this.bus = bus;
  }

  registerAgent(agent: Agent): void {
    this.agents.set(agent.name, agent);
  }

  // 将复杂任务分解并分配给各 Agent
  async decompose(task: string): Promise<TaskAssignment[]> {
    const agentDescriptions = Array.from(this.agents.values())
      .map((a) => `- ${a.name}`)
      .join("\n");

    const response = await openai.chat.completions.create({
      model: "gpt-4o",
      messages: [
        {
          role: "system",
          content: `你是任务分解专家。将复杂任务分配给合适的 Agent。

可用 Agent：
${agentDescriptions}

输出 JSON 数组：
[{
  "agentName": "Agent名称",
  "subtask": "具体子任务描述",
  "priority": "high/medium/low",
  "dependsOn": []
}]`,
        },
        { role: "user", content: `任务：${task}` },
      ],
      response_format: { type: "json_object" },
      temperature: 0.2,
    });

    const parsed = JSON.parse(response.choices[0].message.content!);
    return parsed.assignments || parsed;
  }

  // 并行执行所有子任务，处理依赖关系
  async execute(
    assignments: TaskAssignment[]
  ): Promise<Map<string, string>> {
    const results = new Map<string, string>();
    const pending = [...assignments];

    while (pending.length > 0) {
      // 找出所有依赖已满足的任务
      const ready = pending.filter(
        (a) =>
          !a.dependsOn?.length ||
          a.dependsOn.every((dep) => results.has(dep))
      );

      if (ready.length === 0) {
        console.error("[Orchestrator] 依赖死锁，中断执行");
        break;
      }

      // 并行执行就绪的任务
      await Promise.all(
        ready.map(async (assignment) => {
          const agent = this.agents.get(assignment.agentName);
          if (!agent) {
            console.error(`[Orchestrator] Agent ${assignment.agentName} 不存在`);
            results.set(assignment.agentName, "Agent 不存在");
            return;
          }

          // 收集依赖任务的结果作为上下文
          const contextParts = (assignment.dependsOn || [])
            .filter((dep) => results.has(dep))
            .map((dep) => `[${dep}的结果]: ${results.get(dep)}`);
          const context = contextParts.join("\n\n") || undefined;

          console.log(`[Orchestrator] 分配给 ${assignment.agentName}: ${assignment.subtask.slice(0, 60)}...`);

          await this.bus.publish({
            from: "orchestrator",
            to: assignment.agentName,
            type: "TASK",
            payload: assignment.subtask,
          });

          const result = await agent.run(assignment.subtask, context);
          results.set(assignment.agentName, result);

          await this.bus.publish({
            from: assignment.agentName,
            to: "orchestrator",
            type: "RESULT",
            payload: result,
          });
        })
      );

      // 移除已完成的任务
      for (const a of ready) {
        const idx = pending.indexOf(a);
        if (idx !== -1) pending.splice(idx, 1);
      }
    }

    return results;
  }

  // 聚合各 Agent 结果，生成最终报告
  async synthesize(
    task: string,
    agentResults: Map<string, string>
  ): Promise<string> {
    const resultsSummary = Array.from(agentResults.entries())
      .map(([agent, result]) => `## ${agent} 的分析\n${result}`)
      .join("\n\n");

    const response = await openai.chat.completions.create({
      model: "gpt-4o",
      messages: [
        {
          role: "system",
          content:
            "你是报告综合专家。整合多个专家的分析，生成一份统一、无矛盾的最终报告。",
        },
        {
          role: "user",
          content: `原始任务：${task}\n\n各 Agent 分析结果：\n\n${resultsSummary}\n\n请生成综合最终报告。`,
        },
      ],
      temperature: 0.3,
    });

    return response.choices[0].message.content || "";
  }

  // 冲突解决：当不同 Agent 给出矛盾结论时
  async resolveConflicts(
    topic: string,
    conflictingViews: { agentName: string; view: string }[]
  ): Promise<string> {
    const viewsList = conflictingViews
      .map((v) => `${v.agentName}: ${v.view}`)
      .join("\n\n");

    const response = await openai.chat.completions.create({
      model: "gpt-4o",
      messages: [
        {
          role: "system",
          content:
            "你是仲裁专家。当多个专家意见冲突时，分析各方观点，给出合理的折中或最优解。",
        },
        {
          role: "user",
          content: `议题：${topic}\n\n各方观点：\n${viewsList}\n\n请分析冲突原因并给出建议。`,
        },
      ],
      temperature: 0.3,
    });

    return response.choices[0].message.content || "";
  }
}

完整演示

// src/multi_agent/demo.ts
import { Agent } from "./agent";
import { Orchestrator } from "./orchestrator";
import { MessageBus } from "./message_bus";

async function runDemo() {
  const bus = new MessageBus();
  const orchestrator = new Orchestrator(bus);

  // 创建专业 Agent 团队（对比 CrewAI 的 Agent + role + goal 设计）
  const agents = [
    new Agent({
      name: "架构分析师",
      role: "资深软件架构师",
      goal: "评估代码架构设计，识别设计模式问题和模块化不足",
      constraints: [
        "只关注架构层面，不深入具体实现细节",
        "给出可量化的架构评分（1-10）",
        "列出最多 5 个最重要的架构问题",
      ],
    }),
    new Agent({
      name: "安全专家",
      role: "网络安全工程师",
      goal: "发现代码中的安全漏洞，包括注入、认证、数据暴露等问题",
      constraints: [
        "按 OWASP Top 10 分类安全问题",
        "给出严重程度（Critical/High/Medium/Low）",
        "不修改代码，只提供修复建议",
      ],
    }),
    new Agent({
      name: "性能工程师",
      role: "性能优化专家",
      goal: "识别性能瓶颈，包括算法复杂度、数据库查询、内存使用",
      constraints: [
        "给出量化的性能影响估计",
        "优先级排序：高频路径 > 偶发路径",
        "每个问题提供具体的优化方向",
      ],
    }),
  ];

  agents.forEach((a) => orchestrator.registerAgent(a));

  const task =
    "审查一个电商平台的用户认证模块，该模块包含登录、注册、密码重置功能，使用 JWT token，数据库是 MySQL";

  console.log(`[Multi-Agent] 任务：${task}\n`);

  // 1. 分解任务
  const assignments = await orchestrator.decompose(task);
  console.log(`[Multi-Agent] 分解为 ${assignments.length} 个子任务`);

  // 2. 并行执行（对比 AutoGen GroupChat 的多 Agent 并行）
  const results = await orchestrator.execute(assignments);
  console.log(`\n[Multi-Agent] 所有 Agent 完成，共 ${results.size} 个结果`);

  // 3. 聚合结果
  const finalReport = await orchestrator.synthesize(task, results);
  console.log("\n=== 最终综合报告 ===");
  console.log(finalReport.slice(0, 1000) + "...");

  // 4. 演示冲突解决
  const conflicts = [
    { agentName: "架构分析师", view: "JWT token 应存储在 localStorage" },
    { agentName: "安全专家", view: "JWT token 必须存在 httpOnly Cookie，localStorage 有 XSS 风险" },
  ];
  const resolution = await orchestrator.resolveConflicts("JWT 存储方案", conflicts);
  console.log("\n=== 冲突解决 ===");
  console.log(resolution);

  // 显示消息日志（对比 AutoGen 的对话历史）
  console.log(`\n[Multi-Agent] 消息总数：${bus.getLog().length}`);
}

runDemo().catch(console.error);
