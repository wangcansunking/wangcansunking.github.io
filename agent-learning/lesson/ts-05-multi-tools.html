第5课：多工具并行调用

学习目标

  1. 定义多个工具并让模型自主选择
  2. 理解并行工具调用（一次返回多个 tool_calls）
  3. 统一处理所有工具结果后回传

多工具定义

OpenAI 兼容协议下每个工具用 { type: 'function', function: {...} } 包一层：

import { client, model } from "../shared/llmClient";

const tools = [
  {
    type: "function" as const,
    function: {
      name: "get_weather",
      description: "获取指定城市的当前天气信息",
      parameters: {
        type: "object",
        properties: {
          city: { type: "string", description: "城市名称，如 '北京'、'上海'" },
          unit: {
            type: "string",
            enum: ["celsius", "fahrenheit"],
            description: "温度单位",
          },
        },
        required: ["city"],
      },
    },
  },
  {
    type: "function" as const,
    function: {
      name: "calculate",
      description: "执行数学计算，支持加减乘除和基本数学函数",
      parameters: {
        type: "object",
        properties: {
          expression: {
            type: "string",
            description: "要计算的数学表达式，如 '(15 + 27) * 2'",
          },
        },
        required: ["expression"],
      },
    },
  },
];

并行工具调用

当用户的问题同时需要多个工具时，模型会在一次响应中返回多个 tool_calls：

用户: "北京和上海今天天气怎么样？"

response.choices[0].message:
{
  role: "assistant",
  content: null,
  tool_calls: [
    { id: "call_01", function: { name: "get_weather", arguments: '{"city":"北京"}' } },
    { id: "call_02", function: { name: "get_weather", arguments: '{"city":"上海"}' } }
  ]
}
finish_reason: "tool_calls"

处理多个工具调用

// 模拟工具实现
function executeTools(name: string, input: Record<string, unknown>): string {
  if (name === "get_weather") {
    const city = input.city as string;
    const unit = (input.unit as string) ?? "celsius";
    const temps: Record<string, number> = {
      北京: 22, 上海: 28, 广州: 32, 深圳: 30,
    };
    const temp = temps[city] ?? 25;
    const symbol = unit === "fahrenheit" ? `${temp * 9 / 5 + 32}°F` : `${temp}°C`;
    return `${city}当前天气：晴，${symbol}，湿度 65%`;
  }

  if (name === "calculate") {
    const expr = input.expression as string;
    const result = Function(`"use strict"; return (${expr})`)();
    return `计算结果：${expr} = ${result}`;
  }

  return `未知工具：${name}`;
}

// 取出所有 tool_calls
const message = response.choices[0].message;
const toolCalls = message.tool_calls ?? [];

// 并行执行所有工具（Promise.all 提升效率）
const toolResults = await Promise.all(
  toolCalls.map(async (call) => {
    const args = JSON.parse(call.function.arguments);
    const result = executeTools(call.function.name, args);
    return {
      role: "tool" as const,
      tool_call_id: call.id,                 // 与对应的 tool_call id 匹配
      content: result,
    };
  })
);

回传所有结果，获取最终答案

const finalResponse = await client.chat.completions.create({
  model,
  max_tokens: 2048,
  tools,
  messages: [
    { role: "user", content: userMessage },
    message,                                  // ← 模型的 assistant 消息（含 tool_calls）原样追加
    ...toolResults,                           // ← 每个 tool_result 都是一条独立的 tool 消息
  ],
});

console.log(finalResponse.choices[0].message.content);

关键差异：OpenAI 兼容协议下每个 tool_result 是 messages 数组里独立的 role: 'tool' 消息；Anthropic 原生协议是把所有 tool_result 包在同一条 user 消息的 content 数组里。本课用前者。

完整示例

import { client, model } from "../shared/llmClient";

async function askWithTools(userMessage: string) {
  // 第一次请求
  const response = await client.chat.completions.create({
    model,
    max_tokens: 2048,
    tools,
    messages: [{ role: "user", content: userMessage }],
  });

  const msg = response.choices[0].message;

  if (response.choices[0].finish_reason !== "tool_calls" || !msg.tool_calls?.length) {
    // 模型直接回答，不需要工具
    return msg.content ?? "";
  }

  // 执行所有工具
  console.log(`模型调用了 ${msg.tool_calls.length} 个工具：`);
  msg.tool_calls.forEach((c) =>
    console.log(`  - ${c.function.name}(${c.function.arguments})`)
  );

  const results = msg.tool_calls.map((c) => ({
    role: "tool" as const,
    tool_call_id: c.id,
    content: executeTools(c.function.name, JSON.parse(c.function.arguments)),
  }));

  // 回传结果
  const final = await client.chat.completions.create({
    model,
    max_tokens: 2048,
    tools,
    messages: [
      { role: "user", content: userMessage },
      msg,
      ...results,
    ],
  });

  return final.choices[0].message.content ?? "";
}

const answer = await askWithTools(
  "北京和上海今天天气怎么样？另外帮我算一下 (88 + 12) * 3 是多少？"
);
console.log(answer);

小结

场景 处理方式
单个工具调用 msg.tool_calls[0]
多个工具调用 msg.tool_calls.map(...)，Promise.all 并行执行
回传结果 每个 tool_result 一条独立 role: 'tool' 消息
模型直接回答 finish_reason === 'stop'，不需要工具
