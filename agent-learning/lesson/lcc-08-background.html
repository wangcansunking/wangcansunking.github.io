第8课：Background Tasks — 非阻塞异步操作

介绍

某些任务（构建、部署、长时间测试）执行时间很长，不应阻塞 Agent 主循环。Background Tasks 让 Agent 把这类任务扔到后台执行，同时继续其他工作，之后通过 task_status 查询结果。

源码参考：github.com/shareAI-lab/learn-claude-code — agents/s08_background_tasks.py

遇到什么问题

Agent 遇到长时间操作时：

# 同步执行：主循环阻塞
bash("npm run build")   # ← 阻塞 3 分钟，Agent 无法做其他任何事
bash("pytest tests/")   # ← 阻塞 10 分钟
bash("docker build .") # ← 阻塞 5 分钟

一个处理"构建 + 运行测试 + 生成文档"的 Agent：

# 串行：总耗时 = 3 + 10 + 2 = 15 分钟
build → wait 3min → test → wait 10min → docs → wait 2min → done

# 并行后台：总耗时 ≈ 10 分钟（最长任务决定）
build (bg) ─────────────────── 3min → done
test  (bg) ──────────────────────────────── 10min → done
docs  (bg) ──────── 2min → done

有哪些方案

方案 A：同步 bash（默认）

✅ 简单 ❌ 阻塞主循环，无法并行执行

方案 B：bash + & 后台进程

bash("npm run build &")  # 后台启动，但无法追踪结果

✅ 非阻塞 ❌ 无法获取 exit code、输出难以捕获

方案 C：线程池 + 任务注册表（推荐）

executor = ThreadPoolExecutor(max_workers=4)
futures: dict[str, Future] = {}

def spawn_task(cmd: str) -> str:
    task_id = f"bg_{int(time.time()*1000) % 100000}"
    futures[task_id] = executor.submit(subprocess.run, cmd, ...)
    return task_id

def check_task(task_id: str) -> str:
    future = futures.get(task_id)
    if future.done():
        return future.result()  # stdout + returncode
    return "running"

✅ 非阻塞，可并行多个任务 ✅ 完整的 stdout/stderr/exit code 捕获 ✅ Agent 可随时查询任务状态

怎么选择

方案 C。ThreadPoolExecutor 是 Python 标准库，零依赖。任务注册表让 Agent 能追踪多个并发任务。

怎么执行

后台任务管理器

# agents/s08_background_tasks.py
import subprocess
import time
import threading
from concurrent.futures import ThreadPoolExecutor, Future
from dataclasses import dataclass
from typing import Optional


@dataclass
class TaskResult:
    task_id: str
    command: str
    status: str        # "running" | "done" | "failed"
    stdout: str = ""
    stderr: str = ""
    exit_code: Optional[int] = None
    started_at: float = 0
    finished_at: float = 0

    @property
    def duration(self) -> float:
        if self.finished_at:
            return self.finished_at - self.started_at
        return time.time() - self.started_at


class BackgroundTaskManager:
    def __init__(self, max_workers: int = 4):
        self.executor = ThreadPoolExecutor(max_workers=max_workers)
        self.tasks: dict[str, TaskResult] = {}
        self._lock = threading.Lock()

    def spawn(self, command: str, workdir: str = ".") -> str:
        """启动后台任务，立即返回 task_id"""
        task_id = f"bg_{int(time.time() * 1000) % 100000}"
        result = TaskResult(
            task_id=task_id, command=command,
            status="running", started_at=time.time(),
        )
        with self._lock:
            self.tasks[task_id] = result

        def _run():
            proc = subprocess.run(
                command, shell=True, capture_output=True,
                text=True, cwd=workdir,
            )
            with self._lock:
                result.stdout = proc.stdout
                result.stderr = proc.stderr
                result.exit_code = proc.returncode
                result.status = "done" if proc.returncode == 0 else "failed"
                result.finished_at = time.time()

        self.executor.submit(_run)
        return task_id

    def status(self, task_id: str) -> str:
        """查询任务状态"""
        with self._lock:
            task = self.tasks.get(task_id)
        if not task:
            return f"Task {task_id} not found"

        if task.status == "running":
            return f"[{task_id}] running ({task.duration:.1f}s) — {task.command[:60]}"

        lines = [
            f"[{task_id}] {task.status} in {task.duration:.1f}s",
            f"  cmd: {task.command[:80]}",
        ]
        if task.stdout:
            lines.append(f"  stdout: {task.stdout[:300]}")
        if task.stderr and task.status == "failed":
            lines.append(f"  stderr: {task.stderr[:200]}")
        return "\n".join(lines)

    def list_all(self) -> str:
        """列出所有任务状态摘要"""
        with self._lock:
            tasks = list(self.tasks.values())
        if not tasks:
            return "No background tasks."
        lines = ["=== BACKGROUND TASKS ==="]
        for t in sorted(tasks, key=lambda x: x.started_at):
            icon = {"running": "⏳", "done": "✓", "failed": "✗"}[t.status]
            lines.append(f"  {icon} [{t.task_id}] {t.command[:50]} ({t.duration:.1f}s)")
        return "\n".join(lines)

    def wait_all(self, timeout: float = 300) -> str:
        """等待所有任务完成（可选，用于最终汇总）"""
        deadline = time.time() + timeout
        while time.time() < deadline:
            with self._lock:
                running = [t for t in self.tasks.values() if t.status == "running"]
            if not running:
                break
            time.sleep(0.5)
        return self.list_all()


# 全局实例
bg = BackgroundTaskManager()

工具接口

TOOL_HANDLERS.update({
    "spawn_task":  lambda **kw: bg.spawn(kw["command"], kw.get("workdir", ".")),
    "task_status": lambda **kw: bg.status(kw["task_id"]),
    "task_list":   lambda **kw: bg.list_all(),
    "wait_tasks":  lambda **kw: bg.wait_all(kw.get("timeout", 300)),
})

系统 Prompt

SYSTEM = f"""You are a parallel execution agent at {WORKDIR}.

For long-running operations (builds, tests, deployments):
1. Use spawn_task() to start them in the background
2. Continue with other work immediately
3. Periodically check with task_status(task_id) or task_list()
4. Use wait_tasks() only when you need ALL results before proceeding

This maximizes throughput — run independent tasks concurrently."""

并行执行示例

s08 >> build the frontend, run tests, and generate docs — do them in parallel

# Agent 行为：
# 1. spawn_task("npm run build")    → bg_11234
# 2. spawn_task("pytest tests/")   → bg_11235
# 3. spawn_task("make docs")       → bg_11236
# 4. 继续做其他工作（查看代码、分析结果等）
# 5. task_list() 检查进度
# 6. wait_tasks() 等待全部完成
# 7. 汇报结果

# 输出示例：
# ⏳ [bg_11234] npm run build (45.2s)
# ✓  [bg_11235] pytest tests/ (127.3s)
# ✓  [bg_11236] make docs (18.7s)
