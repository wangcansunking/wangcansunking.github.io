第6课：Reflexion — 自我反思与迭代改进

介绍

Reflexion 让 Agent 能从失败中学习：执行后自我评估，生成反思（reflection），下次尝试时参考反思避免重复错误。普通 Agent 遇到错误只会重试，不会从错误中学习，导致相同错误反复发生。本课实现完整的 Reflexion 框架：Evaluator、Reflector、Memory、带反思的重试循环。

遇到什么问题

普通 Agent 遇到错误只会重试，不会从错误中学习，导致相同错误反复发生：

flowchart TD
    A[任务输入] --> B[第 N 次尝试]
    B --> C[执行]
    C --> D[结果]
    D --> E[Evaluator 评估\n好/差/可接受]
    E -->|满意| F[返回结果]
    E -->|不满意| G[Reflector\n生成反思文本]
    G --> H[Memory\n存储反思]
    H --> I[带反思重试\n避免相同错误]
    I --> B
    B --> J{超过最大次数?}
    J -->|是| K[返回最佳结果]
    style G fill:#fff3cd
    style H fill:#d4edda

有哪些方案

方案 A：简单重试（无反思）

for (let i = 0; i < 3; i++) {
  const result = await execute(task);
  if (evaluate(result)) return result;
}

✅ 简单 ❌ 相同错误反复出现

方案 B：Reflexion（带自我评估和反思）

let reflection = "";
for (let i = 0; i < maxAttempts; i++) {
  const result = await execute(task, reflection);
  const { score, feedback } = await evaluate(result);
  if (score >= threshold) return result;
  reflection = await reflect(task, result, feedback);
}

✅ 每次尝试都比上次更好 ✅ 明确的改进方向，不是盲目重试 ❌ 每次需要额外 2 次 LLM 调用（评估+反思）

方案 C：Reflexion + 持久化记忆

把反思存到文件/数据库，跨会话复用。

✅ 积累的经验可以跨任务使用 ✅ 相同类型的任务越来越好

怎么选择

选择方案 B（Reflexion），对质量要求高的任务额外使用方案 C 的持久化记忆。代码生成、写作、数据分析等需要高质量输出的场景，Reflexion 效果显著。

怎么执行

Evaluator（判断输出质量）

// src/reflexion/evaluator.ts
import OpenAI from "openai";

const openai = new OpenAI({ apiKey: process.env.OPENAI_API_KEY });

export interface EvaluationResult {
  score: number;           // 0-10 分
  passed: boolean;         // 是否达到要求
  feedback: string;        // 具体问题描述
  suggestions: string[];   // 改进建议
}

// LLM-as-Judge：用 LLM 评估输出质量
export async function evaluateOutput(
  task: string,
  output: string,
  criteria?: string
): Promise<EvaluationResult> {
  const response = await openai.chat.completions.create({
    model: "gpt-4o",
    messages: [
      {
        role: "system",
        content: "你是一个严格的质量评审专家。请客观评估任务完成质量。",
      },
      {
        role: "user",
        content: `评估以下任务的完成质量：

任务：${task}
${criteria ? `评分标准：${criteria}` : ""}

输出内容：
${output}

请以 JSON 格式评估：
{
  "score": 0-10 分,
  "passed": true/false（7分及以上为通过）,
  "feedback": "具体问题描述",
  "suggestions": ["具体改进建议1", "建议2"]
}`,
      },
    ],
    response_format: { type: "json_object" },
    temperature: 0.1,
  });

  return JSON.parse(response.choices[0].message.content!) as EvaluationResult;
}

// 代码特定评估器
export async function evaluateCode(
  task: string,
  code: string
): Promise<EvaluationResult> {
  return evaluateOutput(
    task,
    code,
    `
1. 正确性：代码能解决问题，逻辑无误
2. 代码质量：可读性、命名规范、注释
3. 错误处理：有适当的异常处理
4. 效率：算法效率合理
5. 完整性：包含必要的导入和示例`
  );
}

Reflector（生成反思文本）

// src/reflexion/reflector.ts
import OpenAI from "openai";
import type { EvaluationResult } from "./evaluator";

const openai = new OpenAI({ apiKey: process.env.OPENAI_API_KEY });

export interface Reflection {
  analysis: string;        // 失败原因分析
  mistakes: string[];      // 具体错误列表
  improvements: string[];  // 明确的改进方向
  guidelines: string;      // 下次尝试的指导原则
}

// 生成反思
export async function generateReflection(
  task: string,
  attempt: string,
  evaluation: EvaluationResult,
  previousReflections: string[] = []
): Promise<Reflection> {
  const previousContext = previousReflections.length > 0
    ? `\n之前的反思记录：\n${previousReflections.join("\n\n")}\n`
    : "";

  const response = await openai.chat.completions.create({
    model: "gpt-4o",
    messages: [
      {
        role: "system",
        content: "你是一个反思专家，帮助 AI Agent 从错误中学习。",
      },
      {
        role: "user",
        content: `任务：${task}
${previousContext}
本次尝试：
${attempt}

评估结果：
分数：${evaluation.score}/10
问题：${evaluation.feedback}
建议：${evaluation.suggestions.join("; ")}

请深度分析失败原因，提供具体的改进指导，JSON 格式：
{
  "analysis": "失败根本原因分析",
  "mistakes": ["具体错误1", "错误2"],
  "improvements": ["具体改进方向1", "方向2"],
  "guidelines": "下次尝试时应该遵循的原则和注意事项"
}`,
      },
    ],
    response_format: { type: "json_object" },
    temperature: 0.3,
  });

  return JSON.parse(response.choices[0].message.content!) as Reflection;
}

Memory（存储反思供后续参考）

// src/reflexion/memory.ts
import * as fs from "fs";

export interface ReflectionMemory {
  taskType: string;
  reflections: Array<{
    timestamp: number;
    task: string;
    reflection: string;
    score: number;
  }>;
}

export class ReflexionMemory {
  private memory: Map<string, ReflectionMemory> = new Map();
  private readonly persistPath?: string;

  constructor(persistPath?: string) {
    this.persistPath = persistPath;
    if (persistPath && fs.existsSync(persistPath)) {
      const data = JSON.parse(fs.readFileSync(persistPath, "utf-8"));
      this.memory = new Map(Object.entries(data));
    }
  }

  store(taskType: string, task: string, reflection: string, score: number): void {
    if (!this.memory.has(taskType)) {
      this.memory.set(taskType, { taskType, reflections: [] });
    }

    const mem = this.memory.get(taskType)!;
    mem.reflections.push({ timestamp: Date.now(), task, reflection, score });

    // 只保留最近 20 条（FIFO）
    if (mem.reflections.length > 20) {
      mem.reflections = mem.reflections.slice(-20);
    }

    if (this.persistPath) {
      fs.writeFileSync(
        this.persistPath,
        JSON.stringify(Object.fromEntries(this.memory), null, 2)
      );
    }
  }

  getReflections(taskType: string, limit: number = 3): string[] {
    const mem = this.memory.get(taskType);
    if (!mem) return [];

    return mem.reflections
      .slice(-limit)
      .map((r) => `[历史反思 (分数: ${r.score})] ${r.reflection}`);
  }
}

带反思的重试循环

// src/reflexion/reflexion_agent.ts
import OpenAI from "openai";
import { evaluateOutput } from "./evaluator";
import { generateReflection, type Reflection } from "./reflector";
import { ReflexionMemory } from "./memory";

const openai = new OpenAI({ apiKey: process.env.OPENAI_API_KEY });

export interface ReflexionResult {
  finalOutput: string;
  attempts: number;
  finalScore: number;
  reflections: Reflection[];
  converged: boolean;
}

export async function runReflexion(
  task: string,
  taskType: string,
  maxAttempts: number = 4,
  scoreThreshold: number = 7,
  memory?: ReflexionMemory
): Promise<ReflexionResult> {
  const reflections: Reflection[] = [];
  const reflectionTexts: string[] = [];
  let bestOutput = "";
  let bestScore = 0;

  // 从记忆中获取历史反思
  const historicalReflections = memory?.getReflections(taskType) || [];

  for (let attempt = 1; attempt <= maxAttempts; attempt++) {
    console.log(`\n[Reflexion] 第 ${attempt}/${maxAttempts} 次尝试`);

    // 构建包含反思的 prompt
    const reflectionContext = [
      ...historicalReflections,
      ...reflectionTexts,
    ].join("\n\n");

    const systemPrompt = reflectionContext
      ? `你是一个任务执行专家。\n\n根据以下历史反思，避免重复相同的错误：\n${reflectionContext}`
      : "你是一个任务执行专家。";

    // 执行任务
    const response = await openai.chat.completions.create({
      model: "gpt-4o",
      messages: [
        { role: "system", content: systemPrompt },
        { role: "user", content: task },
      ],
      temperature: 0.7,
    });

    const output = response.choices[0].message.content || "";

    // 评估输出质量
    const evaluation = await evaluateOutput(task, output);
    console.log(`  评分: ${evaluation.score}/10 - ${evaluation.feedback.slice(0, 80)}`);

    if (evaluation.score > bestScore) {
      bestScore = evaluation.score;
      bestOutput = output;
    }

    if (evaluation.passed) {
      console.log(`  [Reflexion] 质量满足要求，停止迭代`);
      return {
        finalOutput: output,
        attempts: attempt,
        finalScore: evaluation.score,
        reflections,
        converged: true,
      };
    }

    if (attempt < maxAttempts) {
      // 生成反思
      const reflection = await generateReflection(
        task,
        output,
        evaluation,
        reflectionTexts
      );

      reflections.push(reflection);
      reflectionTexts.push(reflection.guidelines);

      // 存储到记忆
      memory?.store(taskType, task, reflection.guidelines, evaluation.score);

      console.log(`  反思: ${reflection.analysis.slice(0, 100)}`);
    }
  }

  return {
    finalOutput: bestOutput,
    attempts: maxAttempts,
    finalScore: bestScore,
    reflections,
    converged: false,
  };
}

完整演示

// src/reflexion/demo.ts
import { runReflexion } from "./reflexion_agent";
import { ReflexionMemory } from "./memory";

async function runDemo() {
  const memory = new ReflexionMemory("./reflexion_memory.json");

  // 演示1：代码生成任务
  console.log("=== 代码生成（Reflexion）===");
  const codeResult = await runReflexion(
    `实现一个 TypeScript 函数 sortByMultipleFields，
    支持按多个字段排序对象数组，每个字段可以单独指定升序/降序，
    要包含完整的类型定义和使用示例`,
    "code_generation",
    4,
    8,
    memory
  );

  console.log(`\n最终分数: ${codeResult.finalScore}/10`);
  console.log(`迭代次数: ${codeResult.attempts}`);
  console.log(`是否收敛: ${codeResult.converged}`);
  console.log(`\n最终输出:\n${codeResult.finalOutput.slice(0, 500)}...`);

  // 演示2：写作任务
  console.log("\n=== 写作任务（Reflexion）===");
  const writeResult = await runReflexion(
    "写一篇 300 字的产品发布公告，产品是 AI 驱动的代码审查工具，目标用户是中小型软件公司",
    "copywriting",
    3,
    7,
    memory
  );

  console.log(`最终分数: ${writeResult.finalScore}/10，迭代: ${writeResult.attempts} 次`);
}

runDemo().catch(console.error);
