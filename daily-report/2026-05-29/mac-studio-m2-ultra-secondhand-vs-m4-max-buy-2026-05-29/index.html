Mac Studio 选机：M2 Ultra 还是 M4 Max

30 秒速览：4090 之外，有 2-5 万元预算的另一条路

本地大模型这一波，4090 / 5090 + Linux 工作站只是一条主线；另一条主线是 Mac Studio——尤其是统一内存超过 128 GB 的高配档，能跑 Linux 工作站需要多卡才能容纳的稠密 70B / MoE 235B 模型。但 Mac Studio 这条路有它独有的隐形选择题：6 月 WWDC 已经传出 M5 Ultra 要发的消息之后，闲鱼上 M2 Ultra 192GB 二手挂牌价进入 4-5 万元区间；同时苹果中国官网 M4 Max 64GB Mac Studio 起售 24,999 元。同样 2-5 万元预算，国内开发者真实纠结的是：旧一代 192GB 巨内存（M2 Ultra）能跑 Qwen3 235B-A22B 但 token/s 慢、且没有保修；新一代 64GB 内存（M4 Max）跑不动 235B，但 GLM-4.6 Air 与 Qwen3-Coder-30B-A3B 这两档主力国产模型完全 hold 住、还能享受 3 年 AppleCare。

我们手里如果想在 5/29 这周敲定一笔硬件预算，这题不是「哪台更好」、而是「我下一年最常跑哪档模型」决定的。核心论点放最前面：统一内存大小决定能跑多大模型、芯片代决定 token/s 速度；M2 Ultra 192GB 是『以慢换大』、M4 Max 64GB 是『以小换快』，两条路服务的是完全不同的本地大模型工作流。

后面五段把这张表挨段展开。

一、闲鱼 M2 Ultra 192GB：旧芯片配巨内存的现实路径

Mac Studio M2 Ultra 是苹果 2023 年 6 月发布的工作站机型，24 核 CPU、60 或 76 核 GPU（76-core 是 BTO 顶配档）、最高 192 GB 统一内存、800 GB/s 内存带宽。当时新机售价苹果中国大陆 192GB / 8TB 顶配挂过 10 万元；192GB / 1TB 标配约 6 万元。两年过去，二手市场已经把这台机器的价位摁了下来。

闲鱼 5 月份 192GB 配置 M2 Ultra Mac Studio 主流挂牌价进入 4-5 万元区间，知乎技术博主在 2026 年初的分析里写到「M2 Ultra 128GB 二手已经跌穿、192GB 配置 4 万元上下」；少数派 2026 春季 Mac 选购指南里也提到 "Mac Studio M2 Ultra 192GB 二手主流价位 4-4.5 万元"。具体卖家成色、保修剩余、是否带 AppleCare、储存 SSD 容量（1TB / 2TB / 4TB / 8TB）会让价格上下浮动 5,000-10,000 元。一线城市当面交易、本地 90 天三方代购验机服务（闲鱼自营或第三方）能把货物风险压下来。

它的核心价值就一条：192 GB 统一内存是当下消费级硬件里能跑 Qwen3 235B-A22B 全模型权重的最便宜路径。Qwen3 235B-A22B 在 Q3 量化下需要约 100-110 GB 内存，Q4 量化需要 130-140 GB；192 GB 内存留出 50 GB 给 KV cache + macOS 系统是绰绰有余。MacStories 的 Mac Studio 早期 benchmark 笔记里给过 Qwen3-235B-A22B 在 Apple Silicon 上「能跑、是研究机性能档」的结论。

吞吐数字方面，我们能查到的有：M3 Ultra Mac Studio 用 LibraxisAI/Qwen3-235B-A22B-MLX-Q5 跑出 25-35 tok/s；M3 Ultra Q3 量化跑 Qwen3 235B 跑出 19.43 tok/s。M2 Ultra 跟 M3 Ultra 的 GPU 性能差距约 15-25%（M3 Ultra 60/80-core 比 M2 Ultra 60/76-core 强一档），M2 Ultra 192GB 上 Qwen3 235B Q3/Q4 的实际吞吐外推大致是 14-25 tok/s 区间。搜遍 5 月 28 日为止的公开网络，M2 Ultra 192GB + Qwen3 235B-A22B Q3/Q4 MLX 的独立第三方实测数字公开为 0——目前只有 M3 Ultra 数字，M2 Ultra 上是社区外推。

它的代价同样诚实：(1) 已经过保（少数闲鱼卖家剩 3-6 个月 AppleCare）；(2) M2 Ultra GPU 算力比 M4 Max 弱一档但比 M4 Max 多 36 个 GPU 核（76 vs 40）；(3) Apple Silicon 跑 Prompt 阶段（prefill）速度依然是 Mac 整体短板，长文档场景 prefill 等待 30-60 秒不奇怪。

二、新机 M4 Max 64GB：新芯片配主流大小内存的本分选择

苹果中国大陆官网 5/29 当下挂牌的 Mac Studio M4 Max 起售配置：M4 Max 14 核 CPU / 32 核 GPU、36 GB 内存、512 GB SSD，起售 16,499 元。我们关注的「64GB 内存版本」是 BTO 加配档：M4 Max 16 核 CPU / 40 核 GPU + 64 GB 内存 + 1 TB SSD，苹果官网售价 24,999 元，京东 5 月份偶尔有 200-500 元活动券。这一档是当下新机里跑本地大模型的「主流甜点」。

M4 Max 芯片 2024 年 10 月发布，比 M2 Ultra 晚一代半，单核 / 多核性能强一档，内存带宽 546 GB/s（比 M2 Ultra 的 800 GB/s 弱一档但已经够 64 GB 内存饱和读写）。GPU 算力上 M4 Max 40 核大致等同于 M2 Ultra 60 核——这是苹果两代之间 GPU 工艺优化的常见跨代提升。

64 GB 内存的实际位置：能完美跑 GLM-4.6 Air 106B/12B MoE MLX 4-bit 量化版本（约 56-60 GB）、Qwen3-Coder-30B-A3B BF16（约 16 GB）、DeepSeek V4 Flash 284B 跑不动（需要 130 GB+）。Qwen3 235B-A22B 即便 Q2 量化也需要 80 GB+，64 GB 一样跑不动。

吞吐数据方面，5/22 同栏目里 GLM-4.6 Mac Studio M4 Max 256GB 顶配档实测拿过 65-80 tok/s（GLM-4.6 4-bit MLX、Mac Studio 顶配）；64GB 版本算力相同、内存带宽不变，跑 GLM-4.6 Air 4-bit 的吞吐应在同档区间（55-75 tok/s）。Qwen3-Coder-30B-A3B BF16 MLX 在 M4 Max 上的 token 速度通常在 25-40 tok/s 区间，对个人 AI Coding 后端是够用档。

它的代价同样诚实：(1) 64 GB 内存的天花板低，跑不动 200B+ 稠密 / MoE 顶配；(2) 一旦未来出现 「必须 192 GB+ 才能跑」的模型档（社区已经在等 Qwen3-Next-300B-A30B、DeepSeek V5 满血），就只能升级整机；(3) 售价 24,999 元相对一张 5090 GPU（国行 30,000-34,000 元）反而便宜——但要算上「需要一台 PC 主机」这一块。

三、四档模型在两台机器上的真实可达性

把读者真正关心的「我未来一年最常跑的模型在这两台机器上是什么状态」整成一张矩阵：

模型 量化档 内存需求 M2 Ultra 192GB M4 Max 64GB
Qwen3 235B-A22B Q3 / Q4 / Q5 110-150 GB √ 14-25 tok/s（社区外推） ×
DeepSeek V4 Flash 284B/13B MoE Q2 / Q3 130-180 GB △ Q2 紧张可跑 ×
Qwen3.6-35B-A3B BF16 / Q4 70 GB / 25 GB √ 完美 √（Q4 完美）
GLM-4.6 Air 106B/12B MoE Q4 / Q5 56-72 GB √ 慢 √ 完美
GLM-4.5V 多模态 106B/12B FP8 / Q4 100 GB / 65 GB √ FP8 紧张 / Q4 完美 △ Q4 极限
Qwen3-Coder-30B-A3B BF16 / AWQ 61 / 16-18 GB √ 飞快 √ 飞快
Qwen3-VL-30B-A3B 多模态 AWQ / FP8 17 / 30 GB √ 飞快 √ 飞快
MiniCPM-V 4.5 8B BF16 16 GB √ 飞快 √ 飞快
Llama 4 Behemoth 400B（社区猜测） Q3 170 GB √ 极限 ×
DeepSeek V4-Flash 满血 BF16 BF16 568 GB × ×

读这张表的方法：M2 Ultra 192GB 跑得动额外多一档 200B+ MoE / 稠密 70B；M4 Max 64GB 在 70B 以下完全够用 + token/s 更快。如果未来一年你写代码 / 翻译 / 总结 / RAG 这些日常任务，70B 以下完全覆盖、M4 Max 64GB 更合适；如果你做模型对比研究 / 想本地跑顶配开源 / 必须用 Qwen3 235B 与 DeepSeek V4 Flash 级别模型做实验，M2 Ultra 192GB 是唯一接近预算的选项。

特别强调一点：MoE 模型对内存的需求不会因为「激活只有 12B」就降低——MoE 模型推理时所有专家权重都要驻留内存（不然 layer routing 会 page miss），激活只影响算力消耗、不影响内存占用。GLM-4.6 Air 106B/12B 在 M4 Max 64GB 上是 Q4 量化 56-60 GB 紧张能跑、不是 12B 那 8 GB。这一点是 5/22 文章已经摊清的常识，但闲鱼买家这次要再算一遍。

四、闲鱼买二手 Mac Studio 的 6 个实操防坑点

如果决定走闲鱼 M2 Ultra 192GB 这条路，下面是 5/29 当下国内卖家市场的几条硬建议：

  1. 优先「Apple 官网保修剩余 6 个月以上」：闲鱼搜「M2 Ultra 192GB 在保」过滤，价格会高 3,000-5,000 元但风险降一档。在保期间出问题可以走 Apple Genius Bar，过保需要自费换主板（M2 Ultra 主板 + 内存焊死，整机维修 8,000-15,000 元）。
  2. 必须验机：电池循环（针对 MBP 卖家串货，Studio 无电池可跳过）/ Activation Lock / SMC 状态 / 屏蔽罩拆装痕迹。Mac Studio 主要看「是否激活解锁、序列号查保、外观成色」三项。当面交易让卖家用买家 Apple ID 登录、验证 iCloud 激活锁解除。
  3. 储存型号要看清：192GB 内存的 BTO 配置在 SSD 上有 1TB / 2TB / 4TB / 8TB 四档；闲鱼挂牌价不写清楚的话，1TB 与 4TB 差价约 5,000-8,000 元——买家容易踩这个坑。Apple Silicon 是 SSD 焊死的，后期不能换。
  4. GPU 核数 60 vs 76 要问清：M2 Ultra 有 60-core GPU（标配）与 76-core GPU（BTO 加配）两档；本地大模型跑 token/s 跟 GPU 核数线性相关，76-core 比 60-core 快约 25%。卖家挂牌不写的，必须聊天确认。
  5. 避开「M2 Max + 96GB」冒充「M2 Ultra + 192GB」：闲鱼上偶有低价异常挂牌，往往是 M2 Max 96GB 冒充。区分方法：Apple 序列号查 / 关于本机 / sysctl -a 查 hw.model 与 hw.optional.arm.MM。M2 Ultra 是 Mac14,13 或 Mac14,14；M2 Max Studio 是 Mac14,12。
  6. WWDC 6 月份 M5 Ultra 传闻：Macworld 与 MacRumors 2026 春季多源报道，M5 Ultra Mac Studio 大概率 2026-09 至 2026-12 之间发布——这意味着 6 月 WWDC 后 M2 Ultra 闲鱼价位可能再跌 5,000-8,000 元。对买家是好消息，对急用户是「等不起」。

五、新机 M4 Max 64GB 的 4 个增值理由

对应地，如果决定走 M4 Max 64GB 新机这条路，下面是这一档比 M2 Ultra 192GB 更合理的 4 个加分点：

  1. AppleCare+ 3 年保修 + 中国大陆维修网点：京东或 Apple 官网买，1 年标准保修起步、加购 AppleCare+ 1,799 元覆盖 3 年。主板出问题免费换 / 屏幕碎 1,099 元 / 整机摔进水 1,799 元。这一档对单机依赖度高的开发者价值很高。
  2. M5 Ultra 出货之后 M4 Max 仍是「主流配置」：苹果传统是高端机型 Ultra 每两到三年迭代一次，Max 每年一次。M5 Max 大概率 2026 年底跟 MacBook Pro 一起发，但 M4 Max 在生产力市场仍会作为「上一代主流」继续被支持 4-5 年的 macOS 主线更新。
  3. 整机功耗低 + 噪音低：M4 Max 64GB 满载 70-80W，M2 Ultra 192GB 满载 95-105W；夏天家里没空调时，前者风扇基本听不见，后者偶尔会拉高 RPM。家用 / 卧室办公场景这是真实因素。
  4. 保值率 + 可出手性强：新机有发票、有保、有原包装；2-3 年后想出手二手价仍可回收 50-60% 原价。闲鱼 M2 Ultra 192GB 跑两年后再出手，价位可能只剩 2-2.5 万元。

六、5 月份这道选择题的答案：按未来一年最常跑的模型来定

把上面的分析汇成一句话：未来一年你最常跑的本地大模型如果不超过 70B 稠密 / 100B MoE 这一档，新机 M4 Max 64GB Mac Studio 24,999 元是更合理选择；如果你要常跑 Qwen3 235B-A22B 这类 200B+ MoE 模型做研究、对照、SFT，闲鱼 M2 Ultra 192GB 是唯一接近预算的选项。

对国内大部分独立开发者 / 1-3 人小团队来说：

  • AI Coding 后端：GLM-4.6 Air 与 Qwen3-Coder-30B-A3B 是主力——M4 Max 64GB 完美覆盖；
  • 写作 / 翻译 / 摘要：Qwen3.6-35B-A3B / GLM-4.6 Air / Qwen3-VL-30B-A3B AWQ 多模态——M4 Max 64GB 完美覆盖；
  • RAG / 知识库 / 文档 OCR：MiniCPM-V 4.5 + BGE-M3 embedding——M4 Max 64GB 放假档；
  • 模型横评研究 / 顶配开源对比 / 投机解码 draft model 双开：需要 M2 Ultra 192GB 才能进入。

我们手里如果在 5/29 这周必须敲定一笔预算，下面这个决策树可以照走：

  1. 「这一年我会不会本地跑 200B+ 模型？」→ 不会 → M4 Max 64GB 新机
  2. 「我能不能等到 WWDC 后？」→ 能等 → 等 M5 Ultra 价位与 M2 Ultra 二手价位双更新
  3. 「我现在就需要 + 必须本地跑 235B 模型 + 接受过保风险」→ 闲鱼 M2 Ultra 192GB

第四条路是值得提一下的：如果预算能拉到 7-8 万元，2026-03 发的 Mac Studio M3 Ultra 96GB 或 192GB 配置（新机或 6 个月内闲鱼）是更平衡的选项——M3 Ultra 跑 Qwen3 235B Q3 的 19.43 tok/s 已是公开数字，跟 M2 Ultra 拉开 25-30% 性能差距、闲鱼价位 6.5-7.5 万元。这条路适合既要长期用 + 又想跑顶配开源 + 不想再换机的开发者。

我们手里的本地大模型工作流，2026 这一波早就过了「能不能跑」阶段；这一周敲下的硬件决策真正决定的是「未来 12 个月你日常工作里调用 token 的速度跟模型档位」。M4 Max 64GB 与 M2 Ultra 192GB 不是「哪台更好」，是「你跑哪档」——把这条想清楚再下单。
