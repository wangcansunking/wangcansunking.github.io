国产编码套餐怎么选：从智谱包月到 MiniMax 全模态按量

如果你每天用 Claude Code 或 Cursor 写代码，最近大概率被同一个问题反复问到：国产的编码包月套餐这么多家，到底该订哪一个。

先把结论放在最前面：截至 2026 年 6 月初，国产编码套餐已经不是"谁更便宜"这么简单了——它正在从单纯的"编码包月"长成打包文本、语音、图像甚至视频的按量订阅，选型的关键也从"看单价"变成了"看你每天到底写多少、写得多重"。 这篇文章不复述媒体口径，只做一件事：把各家官方定价逐档摆开，告诉你同样一笔钱在哪家买到多少额度、配什么客户端、适合什么人。

一、这场订阅战的真正变化：从"编码包月"到"全模态按量"

过去一年，国内开发者熟悉的是"Coding Plan"这个词——一笔月费，包一个编码模型的固定调用量，接进 Claude Code 当后端用。但到了 2026 年年中，这个形态变了。

最明显的信号来自 MiniMax。它在 2026 年 3 月把原来的 Coding Plan 升级成了 Token Plan（按量套餐），一笔订阅里不再只有编码模型，而是把文本、图像、语音、音乐、视频一起打包进来。智谱、阶跃、Kimi 也各自把套餐拆成了多档，从入门到旗舰拉开了五到十倍的价差。

这背后是一个朴素的逻辑：token 的定价正在走十年前手机流量套餐的老路——从"无限量"慢慢转向分档、计量、按用量买单。对开发者来说，这意味着选型不能再只盯着最便宜那一档，而要先想清楚自己的用量画像。

这张表里有三件事值得先点出来：

  • 价格带高度重合：入门档几乎都压在 ¥29-49/月，进阶档集中在 ¥99-200/月，旗舰档多落在 ¥469-699/月（MiniMax 极速版等个别档会更高）。各家在同一个价位段贴身竞争。
  • 额度口径完全不统一：MiniMax 和阶跃按"次请求"算，智谱按"prompts"算，Kimi 干脆按 token 配额折算成"次"。这让跨家直接比额度变得很难，后面会专门拆。
  • 打包内容差异巨大：有的套餐只给一个编码模型，有的把多模态全塞进来，有的甚至打包了四家不同厂商的模型。

二、五家逐个看：价格、额度、模型对得上吗

下面按家拆开，每一档的价格和额度都以官方定价页为准。价格会随活动变动，所以这里写的是截至 2026 年 6 月初查到的实数。

MiniMax：唯一把多模态一起卖的全模态按量套餐

MiniMax Token Plan 的标准版分三档：Starter ¥29/月、Plus ¥49/月、Max ¥119/月。编码主力模型随版本迭代（6 月初已升到 M3），三档按官方公布的额度，每 5 小时分别给到数百到数千次请求不等，具体以官方定价页为准。

它最特别的地方在于多模态。从 Plus 档起，套餐里就附带了图像、语音的额度；Max 档再加上音乐和视频生成，而且这些多模态额度是独立的，不占用编码模型的配额。这是目前国产套餐里唯一把"写代码 + 出图 + 配音 + 生成视频"装进一个订阅的做法。

一句话定位 MiniMax：如果你不只写代码，还要顺手出图、配音、做短视频，它是唯一一个不用再单独买好几个订阅的选择。

智谱 GLM：档位最清晰、客户端兼容最广

智谱的 GLM Coding Plan 分三档：Lite ¥49/月、Pro ¥149/月、Max ¥469/月。每 5 小时的 prompts 额度分别是约 80、400、1,600 次。可用模型覆盖 GLM-5.1、GLM-5-Turbo、GLM-4.7、GLM-4.5-Air 一整条线。

智谱有两个细节对老用户比较实在。一是额度按 5 小时滚动刷新，用完等下一个周期就行，不会额外扣账户余额。二是客户端兼容做得很广——除了 Claude Code，还支持 Cline、Cursor、Roo Code、Kilo Code、OpenCode、Crush、Goose、Factory 等十余款编程工具，基本你常用的那个都能接。

需要注意一个计费细节：GLM-5.1 和 GLM-5-Turbo 在高峰时段（下午 14:00-18:00）会按 3 倍系数消耗额度，非高峰按 2 倍，限时福利期才是 1 倍。也就是说同样一档套餐，你在什么时间写，实际能写的量会差不少。

一句话定位智谱：档位划分最清楚、模型线最全、能接的客户端最多，是想"接进 Claude Code 当后端"时兼容性最稳的一家。

Kimi：用音乐速度术语命名，额度披露偏模糊

月之暗面的 Kimi for Coding 把四档套餐用音乐速度术语命名，很有辨识度：Andante（行板）¥49、Moderato（中板）¥99、Allegretto（小快板）¥199、Allegro（快板）¥699/月，按年订阅省 20%。核心模型是 Kimi K2.5。

Kimi 的卖点是套餐里附带的不只是编码——还打包了 AI 建站、AI 文档、AI PPT 等会员权益。支持 Kimi CLI、VS Code 插件、Claude Code、Roo Code 等工具，接进 Claude Code 同样顺手。

它的短板是额度披露偏模糊。入门的 Andante 档官方写的是"每 5 小时 token 配额，约 300-1,200 次 API 调用，每 7 天刷新"，但更高档位只写"更大额度"，没给精确数字。这让 Kimi 在跨家比额度时不太好量化，得靠实际试用感受。

一句话定位 Kimi：模型能力在国产编码里属第一梯队，套餐里附带的非编码权益是加分项，适合"写代码之外还想用它做文档、建站"的人。

阶跃 Step Plan：海外比国内还便宜，主打 Agent 场景

阶跃星辰的 Step Plan 分四档：Flash Mini ¥49、Flash Plus ¥99、Flash Pro ¥199、Flash Max ¥699/月，开发者社区有限时 5 折活动，入门价能压到 ¥25/月起。每 5 小时的 prompts 额度分别是 100、400、1,500、5,000 次，目前主力模型是 step-3.5-flash。

阶跃的定位很明确——主打 Agent 和编码场景，所有档位统一用高速推理，不分"标准版"和"极速版"。支持 Cursor、Windsurf、Cline 以及自定义接入。

一个有意思的现象：阶跃的海外定价反而比国内便宜，Flash Plus 在海外是 9.99 美元，国内是 ¥99。这种"墙内外价差"在国产套餐里不算常见。

一句话定位阶跃：价格战打得最猛的一家，社区 5 折后入门档极便宜，适合预算敏感、又想跑 Agent 工作流的开发者。

阿里云百炼：一个订阅打包四家模型，但低档已停售

阿里云百炼的 Coding Plan 走的是另一条路——不是卖自家模型，而是把多家打包。一个订阅里同时装了千问 Qwen3.5-Plus、Qwen3-Coder-Next、智谱 GLM-4.7、Kimi-K2.5 四款编程模型，兼容 Qwen Code、Claude Code、Cline、OpenClaw 等工具。

它当前主推的是 Pro 高级套餐 ¥200/月，新用户首月 39.9 元，每月最多 90,000 次请求（约合每 5 小时 6,000 次）。原来的 Lite 基础版（新用户首月 7.9 元、每月 18,000 次请求）已经在 2026 年 3 月 20 日起停止新购、4 月 13 日起停止续费升级。

一句话定位阿里云：唯一"一个订阅同时用四家模型"的选择，适合想在不同模型间随时切换、不想被单一厂商绑定的人；但要注意低价档已经收掉了。

三、额度怎么换算：为什么"次请求"和"prompts"不能直接比

横评里最容易踩的坑，是拿 A 家的"次请求"去比 B 家的"prompts"。这两个口径根本不是一回事。

来源：各厂商官方文档计费口径说明，截至 2026 年 6 月初，自制对照图

  • MiniMax / 阶跃 / 阿里云按"次请求"计费——一次 API 调用算一次，跟这次调用消耗多少 token 无关。
  • 智谱按"prompts"算——一个 prompt 可能触发多轮模型调用（智谱自己折算过，1 个 prompt 约等于 15 次模型调用）。
  • Kimi底层是 token 配额，再折算成"约多少次调用"给你看。

所以当你看到"智谱 Lite 每 5 小时约 80 prompts"和"阶跃 Flash Mini 每 5 小时 100 prompts"时，别急着下结论说阶跃额度更大——两家对"一个 prompt"的定义、背后模型的吞吐都不一样。

真正靠谱的比法只有一个：按你自己的真实用量去试。 偶尔写几行和每天高强度跑 agent，对额度的消耗是两个量级。一个粗略的参照是，重度使用一天可以轻松吃掉上亿 token，这种强度下入门档很快就会触顶，得直接上进阶或旗舰档。

四、配什么客户端：国产端点怎么接进 Claude Code 和 Cursor

横评里很容易被忽略的一环，是这些套餐到底怎么用起来。国产编码套餐能这么火，一个关键前提是它们大多兼容了开发者已经在用的工具——你不用换编辑器，只要把后端模型换成国产端点就行。

接入方式各家大同小异，核心是三步：

  • 去对应厂商的开放平台拿一个 API Key（智谱 bigmodel.cn、MiniMax 开放平台、Kimi 开放平台、阶跃 platform.stepfun.com）；
  • 在 Claude Code 里把模型的 base URL 和 Key 指向国产端点，或在 Cursor 的自定义模型设置里填同样的信息；
  • 选定套餐里对应的模型名（比如智谱填 GLM-5.1、Kimi 填 K2.5、阶跃填 step-3.5-flash）。

接好之后，你的 Claude Code 工作流完全不变——还是那套熟悉的命令、还是那个终端界面，只是底层换成了国产模型，按你订的套餐额度计费。智谱在这一点上做得最全，官方明确支持 Claude Code、Cline、Cursor、Roo Code、Kilo Code、OpenCode、Crush、Goose、Factory 等十余款工具；阶跃支持 Cursor、Windsurf、Cline；Kimi 支持 Kimi CLI、VS Code 插件、Claude Code、Roo Code。

这里有个实际的小提醒：不同套餐对"一次请求"的额度扣减方式不一样，接进 Claude Code 这种会自动多轮调用的工具时，实际消耗会比你手动单次调用快。所以第一次接国产端点，建议先用入门档跑几天，摸清自己的真实消耗节奏，再决定要不要升档。

五、按开发者类型对号入座

把价格和额度都摆完，最后回到最实际的问题：你是哪种开发者，该落在哪个价位段。按月费分三档对齐，各家的落点是这样的：

来源：各厂商官方定价页，截至 2026 年 6 月初，自制对照图

轻量用户（偶尔写代码、做做小项目）——入门档 ¥29-49 就够了：

  • 只写代码、预算极敏感：阶跃 Flash Mini（社区 5 折 ¥25）或 MiniMax Starter ¥29
  • 还想顺手出图配音：MiniMax Plus ¥49，一个订阅全模态搞定
  • 看重客户端兼容、想接 Claude Code：智谱 GLM Lite ¥49

主力开发者（每天写、有一台主力开发机）——进阶档 ¥99-200 是主战场：

  • 模型能力优先：Kimi Moderato ¥99 或智谱 GLM Pro ¥149
  • 跑 Agent 工作流：阶跃 Flash Plus ¥99 / Pro ¥199
  • 想随时在多家模型间切换：阿里云百炼 Pro ¥200

高强度用户（团队、长时间跑 agent、日耗上亿 token）——旗舰档 ¥469 起：

  • 量大管饱、模型线最全：智谱 GLM Max ¥469
  • 模型能力 + 非编码权益：Kimi Allegro ¥699
  • 要全模态 + 极速推理：MiniMax 极速版 Ultra ¥899

把这些和海外对照一下也很清楚。Claude Code 的 Max 档官方定价远高于国产旗舰档，国产套餐普遍只要它的几分之一价格，就能拿到接近一线水平的编码能力，再接进 Claude Code 或 Cursor 当后端用。对国内开发者来说，这意味着选型空间比一年前宽裕太多——你几乎可以在每个价位段都找到至少三四个能打的选项。

六、回到最开始那个问题

回到开头那个被反复问到的问题：国产编码套餐到底该订哪个。

读完这一圈，答案其实不在某一家身上，而在你自己的用量画像里。这场订阅战真正的好消息是：当各家在同一个价位段贴身竞争、把多模态和多模型都卷进来时，受益最多的是每天敲代码的我们——同样一笔钱，能买到的东西比一年前多了一大截。

价格还会变，活动还会出，新模型也还在路上。但有一件事是确定的：国产编码套餐已经从"能不能用"走到了"怎么挑更划算"。这是一个很舒服的位置——我们这一代开发者，赶上了一个选择多到需要做攻略的时代。把自己的用量摸清楚，剩下的，就是挑一个顺手的开始写。
