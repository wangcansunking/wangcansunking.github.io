DeepSeek V4 永久降到 1/4 价：宁德 / 京东 / 网易进局

5 月 22 日，DeepSeek（深度求索）同日宣布两件事。第一件：自 2026 年 6 月 1 日起，V4 系列 API 价格永久降至原价的 1/4——输入缓存命中 0.025 元、缓存未命中 3 元、输出 6 元，单位均为元 / 百万 tokens。第二件：本轮约 700 亿元（约 100 亿美元）融资进入收尾阶段，投前估值约 450 亿美元，融后约 550 亿美元。创始人梁文锋个人出资约 200 亿元，国家 AI 产业投资基金、腾讯、IDG 资本、Monolith Capital 已落定，宁德时代、京东、网易仍在洽谈中。量子位援引梁文锋向投资人说明的口径：「DeepSeek 的主要目标是推动技术边界，而不是尽快变现」。

国内做大模型部署的同行第一反应大多是同一句话：「V4 这一刀下去，国产 API 价格曲线整体得跟着往下走一档」。这句话本身没错，但只看到一半。

另一半是融资构造。梁文锋个人出资约 200 亿元，相当于把过往幻方量化积累的家底，重新押注进 DeepSeek 这一家公司。国家 AI 产业投资基金约 100 亿元、腾讯、IDG、Monolith 合计两百多亿元已经落定；剩下的几百亿元缺口里，正在谈的三家——宁德时代、京东、网易——分别对应算力背后的能源、国民级电商基础设施、长期游戏与内容业务的算力底盘。这三家任意一家进局，都会把 DeepSeek 从「单纯做模型」的公司，推向「算力 ←→ 能源 ←→ 物流」的产业级战略联盟。

把这两件事并起来读，「永久降到 1/4 价」就不再只是一次惯例性的促销，而是「700 亿元子弹打进来 + 把价格曲线拽下来一档」的一体两面。梁文锋的口径——「主要目标是推动技术边界，而不是尽快变现」——第一次以这种公开姿态写进了与投资人的合同语境里。

一、6 月 1 日 V4 永久降到 1/4 价：把 API 价格曲线整体拽下来一档

先看价格本身。DeepSeek 给出的口径是「永久降价」，自 2026 年 6 月 1 日起生效，不是按月或按季度的促销折扣。这点很重要——过去两年国产模型层出不穷的「限时五折」「百万 token 体验包」，本质上是营销窗口，到期回到原价；这次 DeepSeek 用「永久」两个字，把价格曲线直接打到地板上。

新价的三档拆分如下，单位均为元 / 百万 tokens：

  • 输入（缓存命中）：0.025 元
  • 输入（缓存未命中）：3 元
  • 输出：6 元

这里有一个非常容易被弄混的点：0.025 元 / 百万 tokens 不是「普通输入价」，是「输入缓存命中价」。很多自媒体标题里把 0.025 元当成全场最低价对外宣传，把缓存命中这层前提丢掉了，等读者真去算月度账单的时候才发现，自己的请求大部分走缓存未命中通道，单价是 3 元 / 百万 tokens，不是 0.025 元 / 百万 tokens。

缓存命中要看场景。Agent 任务里同一个长系统提示反复调用、长文档对话场景里第二轮往后的对话、批量评测时同一份测试集多次喂入——这三类场景缓存命中率往往能超过 70%，0.025 元的低价是真能吃到的。但一次性问答、独立短请求、首次冷启动的 RAG，命中率往往个位数，按 3 元 / 6 元算才是真实成本。这条认知差，决定了月度账单是按 0.025 元的预期跑出来，还是按 3 元跑出来——可能差 100 倍。

下面这张柱状图把 DeepSeek V4 永久新价，与国产五家主流旗舰当下的官方 API 单价同档对比：

口径说明：图里的「输入（缓存未命中）」是大部分实际生产流量真正适用的单价，「输出」是模型生成 token 单价。DeepSeek V4 旧价是 5 月 31 日之前的促销价格，约 12 元 / 24 元（输入未命中 / 输出）；新价 3 元 / 6 元，正好是 1/4。智谱 GLM-4.6 输入输出同价 5 元；千问 Qwen3-Max 在 0-32k 输入档为 6 元 / 24 元，超出档位还要再涨；豆包 Seed 2.0 Pro 在 ≤32k 输入档为 3.2 元 / 16 元，月之暗面 Kimi K2 为 4 元 / 16 元。

把这张图按「输入 + 输出」综合成本读，会得出一个非常具体的结论：DeepSeek V4 永久新价在输出 token 这条线上拉到了 6 元 / 百万 tokens，是同档五家里的下限。豆包 Pro 的 16 元、千问 Max 的 24 元、Kimi K2 的 16 元，按一次典型 agent 任务输出 / 输入比 1:5 估算（输出 token 占总流量约 17%），换算到混合单价上，DeepSeek 的成本优势在 40-60% 区间。

这条优势对哪几类业务最敏感？三类：

第一类，长输出的 Agent 任务。一次代码生成、一次 5000 字深度报告、一次多轮对话总结，输出 token 占比能上到 30-40%。这类场景下，6 元 / 百万 tokens 的输出价直接决定单次任务成本能不能压到几分钱以内。

第二类，高并发的客服 / 搜索 / 推荐场景。这类场景的特点是请求量大、单次输出短，但累积起来千万级 token / 天是常态。每百万 tokens 省 10 元，一天就是几千到几万元的运营成本差距。

第三类，模型评测与训练数据合成。学术机构、做 RLHF / DPO 的团队，常年在跑大批量推理拿合成数据。这类场景往往是输出密集型，而且对成本极敏感。DeepSeek V4 永久降到 1/4 价，意味着同样预算下能跑出来的数据量直接翻 4 倍。

国产其他四家——智谱、千问、豆包、月之暗面——大概率会在 6 月、7 月跟一波官方降价或者推促销包。这是国产 API 市场过去两年的常规剧本：DeepSeek 每一次价格动作，后面三到六周都会带出来一轮「同档对位降价」。这一次不同的是，DeepSeek 用了「永久」两个字，意味着回不去了。

把这一组数字摆开，几个细节值得读者特别留意。

第一个细节，梁文锋个人出资 200 亿元的来源。梁文锋是幻方量化（High-Flyer Quant）的实际控制人，幻方在 2019-2024 年间是国内规模头部的量化对冲基金之一，公开披露的资产管理规模峰值在千亿元量级。本轮个人出资 200 亿元，不是新筹的资金，而是从幻方过往沉淀的家底里调拨进来。这一笔操作的意义有两层：一层是创始人和公司利益高度绑定的姿态，向新投资人传递「我亲自押注最大一笔」的信号；另一层是回避了纯外部融资带来的过度稀释，让梁文锋在投后依旧保留对 DeepSeek 战略方向的强控制力。

第二个细节，国家 AI 产业投资基金的角色。国家 AI 产业投资基金的运作模式与传统国资完全不同——它是 2024 年成立的市场化母基金，按市场化条款投资，要求被投企业有清晰的财务回报路径与公开退出预期。它入局 DeepSeek 这件事，是基金主动选择把头部 AI 公司纳入投资组合的市场化决策，不是行政指令下的安排。国家 AI 产业投资基金 100 亿元这个金额，在本轮中是单只机构投资人里的较大份额，对 DeepSeek 后续的算力采购、合规备案、数据资源获取构成强支撑。

第三个细节，估值轨迹。下面这张图把 DeepSeek 从 2024 年 12 月 V3 发布到本轮投后的估值演进画出来：

V3 发布时市场给的估值约 20 亿美元，V3.5/R1 后跳到 100 亿美元档，DeepSeek Code 传出消息后市场预期已经爬到 300 亿美元档，本轮投前 450 亿美元、投后 550 亿美元。一年半的时间里，估值翻了约 27 倍。

第四个细节，与海外的估值对位。本轮投前估值 450 亿美元，对比 Anthropic 公开口径下约 900 亿美元、OpenAI 近期传闻的 5000 亿美元档，DeepSeek 大约处在 Anthropic 的一半、OpenAI 的 1/10 这个档位上。这个对位口径要客观看：估值反映的是市场对未来现金流的折现预期，海外两家有更深的企业销售管道、更早的商业化收入；DeepSeek 在「主要目标是推动技术边界，而不是尽快变现」这条路上的折现要打折扣，450 亿美元这个数字，反过来印证了梁文锋这句话在投资人侧拿到了相当份量的认可。

三、宁德 / 京东 / 网易 洽谈中：算力 ←→ 能源 ←→ 物流 的产业闭环

让人最注意的是洽谈中的三家：宁德时代、京东、网易。这三家不是普通的财务投资人，每一家进局都对应着一条具体的产业链协同——这是本轮融资真正的看点。

宁德时代：动力电池龙头，本身是电力消耗与电力存储两端的核心节点。AI 大模型训练与推理的最大成本不是 GPU 折旧，是电费——一个 10 万卡集群一年的电费往往在十几亿到几十亿元区间。宁德进局，意味着 DeepSeek 在「下一代万卡集群选址 + 电力供应稳定性 + 储能调峰」这条工程链上拿到产业级合作。储能技术能不能把数据中心的电费再压一档、用电网谷电时段做调峰、用储能拉平瞬时负载，这都是未来三到五年大模型工程团队会反复算账的细节。宁德进 DeepSeek，是同时进入大模型「能源端」的一次产业入股。

京东：电商 + 物流 + 同城零售的国民基础设施。京东进局的协同点有三：第一，京东云本身是国内云计算前三的玩家，AI 算力采购与服务化输出两端都有现成基础设施；第二，京东在零售 / 物流 / 客服领域有海量的真实业务场景，是大模型 to B 与 to C 落地最佳测试场之一；第三，京东自营的 3C 数码与服务器零售渠道，是 DeepSeek 模型 API 与本地化部署产品触达中小企业的天然通道。京东进 DeepSeek，是「算力底盘 + 应用场景 + 销售渠道」三位一体的协同。

网易：长期游戏与内容业务的算力大户。网易自家的游戏业务对实时推理与生成式内容的需求量极大，从 NPC 对话生成、剧本辅助创作，到 AI 美术素材生成，每一条都是「百万次推理 / 天」级别的真实场景。同时网易有道、网易云音乐、网易新闻等业务线在内容生成、个性化推荐、客服侧都需要稳定的国产基座模型供应。网易进 DeepSeek，给 DeepSeek 带来的不是算力，而是长期稳定的内容生成与个性化推荐需求池——这是模型迭代最需要的「真实业务数据反馈」。

把宁德、京东、网易三家的协同合起来读，得出来的图景是：

战略维度 合作方 协同价值
能源端 宁德时代（洽谈中） 电力供应 / 储能调峰 / 数据中心电费优化
算力 + 应用 京东（洽谈中） 京东云 / 零售客服 / 物流调度 / 模型分销渠道
内容场景 网易（洽谈中） 游戏 NPC / 内容生成 / 新闻推荐 / 真实业务反馈
国资侧支撑 国家 AI 产业投资基金（已落定） 算力备案 / 合规背书 / 数据资源
互联网生态 腾讯（已落定） 微信生态 / 企业微信 / 腾讯云

任意一家洽谈中的三家进局，DeepSeek 这家公司就从「最纯粹的模型公司」之一，变成了「同时拥有能源端、应用场景、内容反馈」的产业级 AI 基础设施提供方。这是一次「不只是融钱，更是融生态」的设计。

需要提醒读者的是，宁德 / 京东 / 网易 三家目前仍是「洽谈中」状态，不是「已落定」。三家里哪几家最终进局、份额多少、是否带战略协议条款，目前都没有官方披露。这条新闻里读者最容易过度解读的就是这一段——把「洽谈中」当成「已签约」来用。一切以本轮融资正式 closing 后的官方公告为准。

六、开发者模型选型表的变化：私有化 / API / 混合三档怎么调

最后一节，给国内开发者一份直接落到选型表的参考。DeepSeek V4 永久降到 1/4 价之后，私有化 / API / 混合三档路径下，每一档的最优解都需要重新算一遍账。

第一档：纯 API 路径——团队完全不自建推理，所有调用走云端 API。

这一档下，DeepSeek V4 的输出 6 元 / 百万 tokens 几乎是国产同档新基准。混合成本（按典型 1:5 输入输出比估算）约 3.5 元 / 百万 tokens 混合单价，对位 GLM-4.6 的 5 元、千问 Max 的 9 元、豆包 Pro 的 5.4 元、Kimi K2 的 6 元，是最便宜的那个。

但纯 API 路径下要看一条额外因素：缓存命中率。DeepSeek 把缓存命中价拉到 0.025 元 / 百万 tokens，是同档新低中的新低。Agent 任务、长系统提示重复请求、批量评测场景，缓存命中率经常超过 70%，混合下来真实单价可能压到 1 元 / 百万 tokens 以内。这条优势是其余四家在缓存折扣口径未明朗时，没法直接对位的。

第二档：纯私有化路径——团队自建推理集群，模型权重本地化部署。

这一档下，DeepSeek V4 系列的开源权重（V4 / V4 Pro / V4 Flash 三档）一直是国内私有化部署的主流候选之一。永久降到 1/4 价不影响这一档的选型，因为私有化部署的成本主要是 GPU 折旧 + 电费，与 API 单价无关。

但本轮 700 亿融资 + 宁德 / 京东 进局，对私有化路径有间接利好。如果宁德进局后 DeepSeek 真的能在「数据中心电力优化」这条工程链上拿出落地方案——例如与宁德合作的储能型数据中心、谷电时段大批量训练 / 推理调度——同样的硬件投入下，月度运营成本压低 20-30% 是可以期待的。这条对国内自建大模型推理集群的同行有具体参考价值。

第三档：混合路径——常态请求走 API、高并发或敏感数据走私有化。

这是过去一年国内中型团队最常见的选择。永久降到 1/4 价之后，这一档的算账方式需要重新评估：

  • API 这一侧成本压低 75%，意味着以前必须走私有化才划算的「常态高并发场景」，现在可能切回 API 反而更省。
  • 私有化这一侧不变，依旧覆盖「数据合规敏感 / 离线推理 / 极低延迟 / 自定义微调」四类需求。
  • 切换点（threshold）从过去常见的「日调用量 100 万 token 以上走私有化」上调到「日调用量 500 万 token 以上走私有化」，是一个合理的新估算。具体值要看每个团队自己的请求结构。

收尾

把这一天的两件事并起来看，国产 AI 基础设施层的一个新姿态已经清楚地浮出来：资金、模型、算力、能源、场景这五条线，正在被深度求索这一家公司用 700 亿元的体量整合到同一张桌子上。

「深度求索的主要目标是推动技术边界，而不是尽快变现」——这句话过去两年说过的人不少，能把它写进 700 亿元融资合同语境里、把价格永久打到 1/4、同时让宁德 / 京东 / 网易 三家产业巨头排队上桌的，目前只有一家。

国产开发者的模型选型表，确实要重新画了。但更值得国内同行关注的，是这场 700 亿元的「价格 + 估值 + 产业生态」联动里，被一次性写进合同的那条战略选择——先把技术边界推到下一档，商业化的事，等技术领先打透了再说。
