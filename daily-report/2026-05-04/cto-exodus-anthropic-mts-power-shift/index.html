百亿公司 CTO 集体辞职去 Anthropic 当工程师

2025-07 到 2026-03，18 个月里六位百亿美元级公司的 CTO 先后加入 Anthropic，出任 Member of Technical Staff。同一时间窗，Anthropic 估值从 615 亿一路冲到正在谈的 9000 亿美元。这条路径背后，是一线模型权重正在重构整个工程师职级体系。

六个名字、六张时间戳

先把名单摊开。这里每一个名字都不是泛指，都对得上一份公开履历和一条 Anthropic 入职时间戳。

  • Niki Parmar，原 Adept AI 联合创始人 / 高管，2024-12 加入 Anthropic 出任 MTS（2025-02 公开宣布）。她是 Transformer 奠基论文「Attention Is All You Need」八位共同作者之一，加入后参与了 Claude 3.7 Sonnet 的研发。
  • Henry Shi，Super.com 联合创始人 / CTO，2025-07 加入 Anthropic 出任 MTS。Super.com 是一家年化收入做到 2 亿美元的旅行 + 金融 SaaS 独角兽，他在花了 9 个月平稳交接之后转身。
  • Box CTO（姓名公开报道未明确披露），2025-12 加入 Anthropic 出任 MTS。Box 是企业内容协作领域的上市公司，市值约 50 亿美元。
  • Mike Krieger，Instagram 联合创始人 / 原 CTO，2024-05 加入 Anthropic 出任 CPO，2026-01 转岗 Labs 团队，与团队共同主管 Claude 前沿能力孵化。Krieger 这一条严格说不是「CTO 直接转 MTS」，而是「CPO 进 Labs 跟工程团队坐到一桌」，但他出现在这份名单里，原因是同一类的取舍——离一线模型更近。
  • Bryan McCann，You.com 联合创始人 / CTO，2026-03 加入 Anthropic 出任 MTS。Salesforce Research 出身，在 NLP 深度学习领域贡献过 CoVe、decaNLP 等被引用上千次的论文。
  • Peter Bailis，Workday CTO，2026-03 加入 Anthropic 出任 MTS，主攻强化学习工程。他原本是 Stanford CS 助理教授，DAWN 项目联合负责人；创办 Sisu Data 融资 1.28 亿美元后被 Snowflake 收购；之后在 Google Cloud 做企业数据 AI 产品，再去 Workday 当 CTO 主导 agentic AI 战略。

这六个人有几个共同点值得记住。第一，他们不是普通工程师转 IC，全部带着完整的研究履历——Stanford CS 教师、Salesforce Research、Adept 联创、Transformer 论文作者；第二，他们都不是「年轻人想体验前沿」，而是已经做过一轮甚至两轮成功创业 / 上市公司高管之后，主动选择在工程师职级表上「向下走一格」；第三，他们去的同一个目的地——Anthropic，而不是分散到 OpenAI、Google DeepMind、Meta Superintelligence Labs 几家之间。

MTS 这个职位在 Anthropic 不是降级，是默认

「Member of Technical Staff」一词在 Anthropic 和 OpenAI 都是工程 / 研究统一称谓。OpenAI 总裁 Greg Brockman 公开解释过这个设计——前沿实验室不愿意把人「分类成研究员和工程师」，所以在外部职级表上看起来「平」，但里面的人都直接接触模型训练。

这套设计有它的工程意义：训练一个万亿参数模型，需要 RL 工程师、infra 工程师、数据工程师、研究员、产品工程师在同一个 PR 上来回。如果他们职级隔三层 manager，等审批走完，模型已经迭代两版。把所有人都叫 MTS，不是没有职级，而是不让职级阻挡 PR 链路。

回看名单里 Niki Parmar 这条——她是「Attention Is All You Need」八位共同作者之一，2024-12 加入 Anthropic 之后直接进入 Claude 3.7 Sonnet 的训练组。从外部职级看，她比 Anthropic 大部分管理层「资历都老」，但内部职位仍然是 MTS。这个安排传递的信号很明确——研究履历越深的人，越愿意接受这个统一称谓，因为它代表的不是降级，而是「你不需要管 30 个人也能影响下一版模型」。

公开访谈里多位 Anthropic 现任员工提到一个共同感受——中级 IC 也能影响高层级技术决策。一位 Workday CTO 这一档的资深工程师，到 Anthropic 当 MTS，意味着他和 Dario Amodei、Tom Brown 在同一张训练计划文档上签字，而不是隔着 4 层 VP 给方向。

但要注意一个新的拐点：Anthropic 早期那种纯扁平结构，已经在调整。 公开员工评价里提到，Anthropic 雇员到 1500 人后，开始增设 VP 层，组织正在向更传统的层级靠拢。也就是说，2025 年加入的 MTS 享受到的扁平红利，会比 2027 年加入的更多。这一波 CTO 转身的窗口期，本身可能就是一两年。

Anthropic 估值 9000 亿，是这件事的金融底色

把这六张时间戳放回 Anthropic 估值曲线上看，会非常清楚。Anthropic 从 2025-03 的 615 亿美元估值起步，2025-09 Series F 时到 1830 亿，2026-02 涨到 3800 亿，4 月底正在谈 9000 亿美元的新一轮、约 500 亿美元规模的融资。同一时间点 OpenAI 在前一轮 Series F 拿到 8520 亿美元估值。

这张表的意义不是「Anthropic 估值大」，而是这六位 CTO 在原公司的股权 + 加入 Anthropic 时的入职 package，加在一起的金融体量足够说服他们「换个职位重要」。Anthropic 年化收入已经到 300 亿美元，9000 亿目标估值如果拿下，就是当今全球最高估值私营公司。一位顶尖工程师在这种增速曲线里拿到的 RSU，单年净增可以覆盖原公司 CTO 三五年薪资。

把这件事写成「他们图的是金钱」当然简化了——更准确的说法是，当一家公司既能给你最高级别的金融回报、又能给你离一线模型最近的工程位时，「头衔」这个变量就被消掉了。Henry Shi 自己在公开发言里讲得很直白：「如果 AGI 在 2027 或 2028 年到来，我将置身前沿实验室，坐在前排观看。」

Anthropic vs OpenAI：用人哲学差异

外界容易把 Anthropic 和 OpenAI 看成「同一种公司，规模略有差异」。但 2026 这一波 CTO 流向告诉我们，两家在用人哲学上的差异其实很大。

Anthropic 的 IC 通道权重，仍然显著高于 OpenAI 的产品向团队。Anthropic 公开宣称的核心价值是 social-impact 和 flat hierarchy，研究优先于层级决策；这套文化的工程结果就是中级 IC 的代码 / 模型方案能直接进训练。OpenAI 早期也是这个调子，但 ChatGPT 商业化之后，产品节奏被几亿用户外驱动，IC 自主性相对减弱。

第二个差异是新近高管来源不同。Anthropic 这一波吸引的，主要是「企业 SaaS 出身的资深 CTO」——Workday、Instagram、Box、You.com、Super.com 都是产品 / 商业化做到很成熟的公司。OpenAI 这两年的高知名度新近加入者，更多是 Google Brain / DeepMind 系研究员（包括 Noam Shazeer 等）。换句话说，Anthropic 在「补企业 + 产品工程实战」，OpenAI 在「补研究纵深」。

第三个差异是对外姿态。OpenAI 的对外脸孔是 ChatGPT 这个 C 端产品 + 频繁发布会；Anthropic 的对外脸孔是 Claude API + 企业协议 + Constitutional AI 论文。对要离开 CTO 转 IC 的人来说，Anthropic 这种「让产品去对外，让工程师对内做模型」的分工，恰好是他们想要的——不再被发布会日程牵着走。

国内对位：模型一线工程师梯队怎么搭

把视角拉回国内。国内 AI 同行最近 18 个月发生过多少类似的「资深 CTO 转模型一线 IC」？答案是——有，但通道还在搭建中，节奏比 Anthropic 略慢。

国内目前能给「资深工程师 / 前管理者」提供「离一线模型最近的位置」的公司，主要是 DeepSeek、千问（Qwen）/ 阿里通义、字节豆包、智谱、月之暗面（Moonshot/Kimi）、美团 LongCat、阶跃、面壁、百川、上海 AI 实验室 这一类。这些团队的核心训练 / 后训练岗位，事实上就承担了对位 Anthropic MTS 的角色。

公开报道里我们看到的国内对位案例：

  • DeepSeek 团队整体偏研究型组织，很多核心成员是清北 / 港科大博士直接进核心训练组，履历上没有「先做几年管理再转 IC」这一段，因为团队本身就是 IC-first 设计。
  • 美团 LongCat 2026 上半年首次披露用 5 万张国产卡训出万亿参数模型，公司层面把"模型一线"作为基础投资，跨界从外卖体系投入万亿级训练。
  • 字节豆包 / 阿里通义两家有更传统的层级结构，从 P/层级体系到大模型核心组的转岗通道存在，但需要内部走流程；外部高知名度 CTO 横跳进来直接当一线训练工程师的案例较少见诸公开报道。
  • 国内工程师进海外前沿实验室也是另一种形态的「离一线模型近」——直接进入海外公司从外部接触一线模型。最近的公开案例是一位中文工程师在 OpenAI 主导中文图像渲染优化。这条路过去几年比较多，未来一两年会持续。
  • 国内大厂内部跨部门进训练组也有发生，例如阿里多名云 / 中间件资深工程师转岗到通义大模型团队、腾讯多位游戏 / 微信 AI 团队工程师集中到混元核心训练组。这种内部流动比海外的「百亿公司 CTO 跨厂转 MTS」摩擦低很多——签证、家庭、税务都不用动，纯职位选择。

国内的事实是，「IC 通道」这个概念正在被越来越多 AI 公司明确制度化。阿里多年前就有 P 序列技术专家通道（P9-P12 资深研究员档），DeepSeek 类研究型组织从一开始就把 IC 当作主路径，字节也有 1-1 至 5-1 的层级体系并配套技术专家轨。国内不缺 IC 通道，缺的是「百亿公司高管主动转 IC」这种叙事——因为国内大部分 AI 大厂的核心训练组本身就是高密度博士 + 资深研究员，没有「外部高管降级进来」这个 motion。

这一点反过来也说明国内的人才结构起步就更接近 Anthropic 的最终形态——研究密度高 + 管理层级薄。Anthropic 是从扁平往传统层级长，1500 人后增设 VP 层；DeepSeek、Moonshot、智谱、阶跃这一档国内研究型组织，本身就是从研究优先开始。两条路径会在中段交汇。

国内 35 岁工程师的两条通道

把这件事再往下拉一层，对国内一线 AI 开发者意味着什么？

国内一线 AI 工程师过去几年经常面临的选择是「往管理走还是往技术走」。35 岁前后这道关口尤其明显——往管理走，要承担更大的人事 / KPI 压力；往技术走，要在职级表上接受「同年龄段晋升节奏会慢一截」的现实。

Anthropic 这一波给国内同行的启发，不是「赶紧投简历去美国」——海外签证、家庭、文化适配各种现实约束都在，且大部分国内开发者的最大价值在于深耕国内场景。真正的启发在于：当海外把"离一线模型最近的位置"明确做成最高权重的工程位时，国内 AI 公司的人才设计也会随之调整。

我们已经能看到的国内信号：

  • DeepSeek、Moonshot、智谱这一档研究型公司，公开招聘里把"参与万亿模型训练 / 后训练 / RL"明确写成最核心岗位，待遇对标海外大厂 IC，没有强制要求管理经验。
  • 阿里通义、字节豆包、腾讯混元这一档大厂内部，技术专家通道近年逐步透明化，资深 IC 的薪酬包向资深 manager 靠拢。
  • 上海 AI 实验室、之江实验室等国家队 + 头部高校研究组，是另一个对位选择，给的不是高估值股权，而是稳定的研究环境 + 一线训练算力。

如果你是国内一线 AI 开发者，不必照搬 Anthropic 这条路径，可以参考这套判断框架——把眼前选择拆成两个变量：

  1. 离训练一线的距离——你写的代码 / 调的参数 / 设计的数据流，能不能直接进下一版基础模型？还是隔着多层 manager 才能影响方向？
  2. 算力 + 数据的可见度——你能不能直接看到自己负责模块的 GPU 时长 / 数据规模 / 评估曲线？还是只能看到几个汇报指标？

这两个变量都打高分的位置，无论职位名是「研究员」「IC」「技术专家」「P10」「6-3」还是「MTS」，本质都是同一类——离一线最近的工程位。中国正在快速建立这类位置，节奏比五年前明显加快。

写在最后：给国内同行的一份判断

这一波 CTO 流向 Anthropic 不是「美国对中国的人才优势」，而是「前沿实验室对传统组织的工程位优势」。这个优势在哪个国家都成立——只要你的国家有前沿实验室。

中国正在长出来的前沿实验室——DeepSeek、Moonshot、智谱、阶跃、面壁、阿里通义、字节豆包、美团 LongCat、上海 AI 实验室——人才设计也会同向收敛。我们这一代国内 AI 开发者其实赶上了一个非常好的时间窗：进 IC 通道不再被视为「不上进」，而是被视为「离价值最近」。前辈已经把通道趟出来了，年轻同行可以更从容地选——选研究型组织，还是选大厂技术专家轨，还是选海外前沿实验室——三条路都通，不必焦虑。

管理多少人不再是衡量价值的唯一尺子。离模型多近，正在变成新的尺子。 中国的尺子，会按中国节奏立。一起加油。
