支持Claude/Cursor/Copilot：一个命令将任何代码转为可问答的知识图谱

ai-daily · 2026 年 5 月 23 日 19:25 · 来源：GitHub Trending any

刚看到一个让我愣神的东西——有人把读代码这件事，从「硬啃」变成了「逛街」。

一个叫 Understand-Anything 的开源项目今天冲上了 GitHub Trending。它做的事情很简单：把你扔进一个 20 万行的陌生代码库，然后给你一张可以点、可以搜、可以问的交互式知识图谱。

不是那种花里胡哨的依赖图（它的 README 原话：Graphs that teach > graphs that impress），而是每个文件、每个函数、每个类都变成一个节点，点一下就能看到纯英文解释、上下游关系，甚至自动生成一条按依赖排序的导览路线。最狠的是那个 /understand-diff 命令——你改代码之前，它能告诉你这次改动会波及系统的哪些部分，相当于给每次 commit 装了个「蝴蝶效应」预览器。

支持的平台多到离谱：Claude Code、Cursor、Copilot、Gemini CLI、Codex，一条 curl 命令全平台通用。我试了一下那个在线 demo，拖动图谱的感觉像是在逛一座属于你代码的博物馆——只不过导览员是个 AI。
