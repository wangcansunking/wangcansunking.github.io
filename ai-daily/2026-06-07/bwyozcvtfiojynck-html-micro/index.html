Anthropic凭Claude Code逆袭：9650亿估值背后的编程革命

如果你把 2026 年上半年大模型赛道的关键变量拉出来做一次回归，会发现参数量、月活、多模态这些曾经被投资人反复咀嚼的指标，在统计上几乎解释不了任何估值变动。真正在散点图上拉出一条陡峭斜线的，只有一个东西：Coding 跑分。

中国这边，DeepSeek 正在谈一笔 70 亿美元的融资，一旦落地就是国内 AI 公司史上最大单笔，估值直奔 590 亿美元。月之暗面在 Kimi K2.5 把 Coding 能力拉满之后，20 天内拿到的收入超过了 2025 全年，ARR 三个月冲到 2 亿美元，半年四轮融资合计超过 39 亿美元，估值飙到 200 亿美元。智谱 GLM-5 在 SWE-bench Verified 上拿下开源模型榜首，MaaS 平台 ARR 同比暴涨 60 倍，港股市值一度触及 8800 亿港元。而 MiniMax——此前最“不 Coding”的那一家——港股上市首日市值突破 1000 亿港元，但后期增长弹性明显偏低，直到三天前扔出 M3，在 SWE-bench Pro 上超越 GPT-5.5、逼近 Claude Opus 4.7，才打出了自己最有力的一拳。

这条从 Coding 跑分到估值倍数的映射关系，已经不需要任何统计检验来证明。

资本市场的定价模型里只剩下一行公式：谁写好代码，谁就值钱。

真正把这件事钉进全球资本市场信仰体系的，是 Anthropic。

2026 年 5 月 28 日，Anthropic 完成 650 亿美元 H 轮融资，估值达到 9650 亿美元，正式超越 OpenAI 的 8520 亿美元，坐上了全球 AI 估值第一的宝座。这个数字放在一年前，几乎没有人敢预测。2025 年底，Anthropic 的 ARR 还只有 90 亿美元，不到 OpenAI 同期 200 亿美元的一半。但仅仅五个月后，它的 ARR 飙到了 470 亿美元，把 OpenAI 的 250 亿美元甩开一个身位。2026 年 Q1 单季营收 48 亿美元，Q2 预计破百亿并有望首次做到盈利——如果兑现，Anthropic 将成为全球第一个赚钱的 AI 独角兽。

它做对了什么？答案可以压缩到两个词：Claude Code。

2026 年 2 月，Claude Opus 4.6 和 Sonnet 4.6 先后发布。100 万 Token 上下文窗口加上多 Agent 并行协作能力，把端到端全栈 Coding 重构变成了一种可以流水线操作的东西。Sonnet 4.6 用 Opus 六成的成本逼近其编程能力，让 Claude Code 从“编程辅助工具”彻底升级为可规模化部署的生产力基础设施。据雷锋网报道，截至 2026 年 2 月，仅 Claude Code 这一条产品线就单独贡献了 25 亿美元 ARR。

当“Chat”只是皮肤，“Coding 执行”才是肌肉

4 月 Ramp 报告披露了一个让 OpenAI 高管失眠的数据：在企业端付费率上，Anthropic 以 34.4% 完成了对 OpenAI 32.3% 的反超。而且这个反超是在综合成本更高、甚至主动限速的情况下做到的。

这意味着什么？OpenAI 手里有 ChatGPT 近十亿周活跃用户，有全球最强的 C 端品牌认知，有一整套从文本到图像到视频的多模态产品矩阵。但它输给了一家专注做 Coding 的公司。不是输在用户量上，而是输在执行能力上。企业客户不关心你聊天聊得多顺滑，他们关心的是你写的代码能不能直接跑通、能不能调试、能不能部署。Chat 只是那层皮肤，Coding 执行才是真正的肌肉。

这个认知一旦确立，整个行业的叙事框架就被击穿了。过去两年大模型公司都在疯狂找 PMF，找下一个杀手级应用，在巨大的不确定性中烧钱试错。现在答案摆在桌上，简单到令人不安：历史上最大的软件需求，就是写软件本身。全球每年软件工程支出超过 1 万亿美元，而一个能把 Coding 做到极致的模型，不是在抢某个细分应用的市场——它是在抢整个软件开发工作流的基础层。

对于大模型公司而言，你可以不做“龙虾”这类爆款应用，可以不讲花哨的故事，你只需要把 Coding 做到最强，然后作为基础设施向无数下游应用和企业工作流输出 API。谁写好代码，谁就是水电煤，谁就卡住了生态链条里话语权最重的那一环。

Anthropic 把这个逻辑跑到了极致。ARR 从 90 亿到 470 亿，五个月。从不到 OpenAI 的一半到彻底反超，四个月。估值翻 2.5 倍，三个月。资本市场的信号再清楚不过：Coding 能力已经不再是一项技术指标，它就是估值最核心的驱动因子。

中国牌桌：从 DeepSeek 到 MiniMax，所有人都看懂了

中国市场的变化同样令人咋舌，而且节奏更快。

DeepSeek 的 70 亿美元融资谈判，590 亿美元估值，背后是它在 Coding 能力上建立的口碑。月之暗面在 Kimi K2.5 上的投入，几乎是一次 all-in Coding 的战略赌博——结果 20 天收入超全年，ARR 三个月冲到 2 亿美元，估值半年翻几倍。智谱 GLM-5 在 SWE-bench Verified 开源模型榜首、GLM-5.1 在 SWE-bench Pro 登顶全球第一，直接推动了港股市场对它重新定价，市值一度触及 8800 亿港元。

最有意思的其实是 MiniMax。这家公司此前最“不 Coding”，上市首日市值 1000 亿港元，但后期增长弹性明显不如同行。直到三天前，MiniMax 扔出了 M3——在 SWE-bench Pro 上超越 GPT-5.5，逼近 Claude Opus 4.7，并同步推出 Coding 智能体 MiniMax Code。一个曾经离 Coding 最远的中国公司，现在打出了它最有力的一拳。这说明所有人都看懂了这场牌局的规则，而且追赶起来并不慢。

模型迭代曲线还远未走平。DeepSeek、Kimi、智谱 GLM、MiniMax、Anthropic、OpenAI——所有人的努力最终指向同一个终点。Coding 跑分越高，API 调用量越大，收入增长越快，估值越贵。过去你可以差异化，走 C 端流量，做政企私有化部署，现在这些策略都只是从 Coding 能力这根主干上分岔出去的枝。主干不够粗，一切都不够牢固。

牌桌上的规则从来没有像现在这样残酷，也从来没有像现在这样简单。写代码这件事，曾经是软件工程师的饭碗，现在成了大模型公司争夺定价权的终极战场。谁能想到，估值近万亿的战争，最后比的是谁的模型更会写代码——这大概是 2026 年最优雅也最讽刺的注脚。
